# Code of Conduct

Coyote3 follows a documented Code of Conduct to keep communication respectful, inclusive, and constructive.

## Scope

The Code of Conduct applies to:

- project issues and pull requests
- chat/discussion channels
- release and operations collaboration

## Expectations

- Be respectful and professional.
- Give actionable, technical feedback.
- Assume positive intent, but call out harm clearly.
- Keep discussions focused on outcomes and safety.

## Reporting

If behavior violates these expectations, use the escalation path defined in the repository root `CODE_OF_CONDUCT.md`.

## Canonical file

The authoritative policy text is in `CODE_OF_CONDUCT.md` at repository root.
